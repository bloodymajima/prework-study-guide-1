HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the title for the webpage.
  • The style element allows you to stylize your elements, inclduing color, backgrounds, borders, and font-size..
  • The body element defines the base URL for website.
  • The link element connects an external resource to the webpage.
  • The meta element defines metadata such as character set, description, keyword, author, and view point.
  • The script element embeds client side scripts in the HTML document.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Using h1 and h2 lets you center a title or heading on webpage.
  • Using commas (ex. header, footer) allows you to affect more than one element that share the same styling.
  • The img selector assigns property values to any image element in the document.
  • The ul selector declares the size of elements on the document.
  • You must link your CSS file to your HTML document to be able to apply the changes you made to the elements.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages file
  • git commit -m "description": creates a snapshot of currently staged changes, and updates starter code
  • git pull origin main: pulls code from the most relevant version of project
  • git push origin "branch-name": ensures you have latest version of project on local computer

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An if statement is a way of introducing decision making into our code. Code will only execute if the statement meets a condition, or is true.
  • Conditional statements allow us to interrupt control flow.
  • An array is a single variable that is used to hold a group of data. Typically, it is used to hold data that is related.
  • A for loop uses a predictable pattern of indexes to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • A for loop contains three important statements. The starting point: declaring the variable x and give it the value of 0. The condition: as long as the number in the variable x is less than the length of the array, the loop will keep running. And the final statement: what allows the array to iterate over each time.
  • A function is simply a set of instructions that tells the computer how to preform a task.
